# Portfolio - Next.js

My Portfolio 
Latest Updated on : 01 Feb 2026

## Features

- **Framework**: Next.js 15 with App Router
- **Styling**: Tailwind CSS
- **Theme**: Minimal black & white developer theme
- **Fonts**: Inter (sans-serif) and JetBrains Mono (monospace)
- **Pages**: Home, About, Projects, Skills, Resume, Contact
